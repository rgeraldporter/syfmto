# SYFMTO
### v0.1.0

Syfmto (["sometimes you fail more than once"](http://robotlolita.me/2013/12/08/a-monad-in-practicality-first-class-failures.html#sometimes-you-fail-more-than-once)) is a Fantasy-land compatible Validation based upon Ramda. Most of this code was adopted from [ramda-fantasy-validation](https://github.com/mrkm4ntr/ramda-fantasy-validation) and adjusted to work better in the browser alongside Ramda.

## Status

This is a work in progress.
